Group Control — Make new directory
style="background-color: #00ff00;" />

New directory:
(Letters and numbers only, please. Anything else will be removed.)

Restricted access: Access control list will be set up. Users listed in directoryname_read will have read access, users in directoryname_write will have write access. Users in directoryname_admin will have admin access, which means they can modify the user lists. This is the default setting.

All valid users have access: This means that any user who has a valid account can have read access. Write and admin access work as in "restricted access".

New directories start with no one in access control lists, and only the creator of the directory in the admin list. New directories are set to restricted access.
